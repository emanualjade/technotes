Pub-Sub

  • Lots of browser-level Javascript relies on events. Clicks, change, blur, etc
  • What about javascript logic? Pub-Sub bridges this gap
  • A piece of code will raise an event
  • N number of pieces will react to it
  • Allows for disconnected communication between function
